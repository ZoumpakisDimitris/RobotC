# robotC

C code solutions for LEGO MINDSTORMS EV3 robotics challenges.  
Each folder contains a challenge with the corresponding C source code and explanations.

---

## Overview

This repository contains solutions for various LEGO EV3 robotics challenges, demonstrating sensor usage, motor control, and algorithmic problem-solving in C. Each source file includes comments explaining the hardware setup, logic, and control flow.

---

## Challenges

- **85cm Challenge** – Navigate a set distance with precise movement.  
- **ArmMotorPosition** – Control the robotic arm to specific positions.  
- **Cargo Retrieval** – Pick up and transport objects.  
- **Color Sensor Comparison** – Use color sensors to distinguish surfaces.  
- **Dizzy Drill** – Rotate or spin maneuvers with stability control.  
- **Forward Until Touch** – Move forward until a sensor detects an obstacle.  
- **Loop with Counter Control** – Demonstrate loops using counters.  
- **Looped Movements** – Combine multiple movements in a loop.  
- **Orchard Challenge** – Navigate through a simulated orchard layout.  
- **Square Dance / Square Dance 2** – Program repeated geometric patterns.  
- **Threshold** – Respond to sensor input thresholds.  
- **Turn for Angle Part 1 & 2** – Rotate the robot accurately to specific angles.  
- **Sensabot Classic** – Classic line-following or maze-solving logic.  
- **Vacuum** – Simulated cleaning behavior with motors and sensors.
